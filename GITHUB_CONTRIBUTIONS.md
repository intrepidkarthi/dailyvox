## How to Submit a PR

For every repo below, follow this process:

1. **Fork** the repository
2. **Read** CONTRIBUTING.md (if it exists) — follow their format exactly
3. **Find** the right section for DailyVox
4. **Add** your entry matching the existing format (copy a similar entry and modify)
5. **Write** a clear PR title: "Add DailyVox — offline AI voice journal for iOS"
6. **Write** a PR description explaining why DailyVox fits the list criteria
7. **Submit** the PR and respond to any reviewer feedback within 24 hours

**Common entry format:**
```markdown
- [DailyVox](https://getdailyvox.com) - Free, offline AI voice diary for iPhone. 100% on-device processing, Digital Twin personality model, mood tracking. No data collection. ([App Store](https://apps.apple.com/app/id6760454642))
```

**Tips:**
- Don't submit to all 52 repos at once — do 3-5 per week to avoid looking spammy
- Personalize each PR to match the repo's focus (privacy angle for privacy repos, AI angle for ML repos, etc.)
- If your PR is rejected, don't resubmit — ask for feedback and adjust
